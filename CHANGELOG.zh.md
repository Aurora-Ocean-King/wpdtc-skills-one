## 1.0.0 - 2026-05-11

### 首次发布

wpdtc-skills-one 首次发布，包含 18 个 AI 驱动的技能，涵盖内容生成、发布和工具类工作流。

**内容技能:**
- `wpdtc-article-illustrator` - 智能文章插图，类型 x 风格 x 色板三维系统
- `wpdtc-comic` - 知识漫画创作器，画风 x 基调灵活组合
- `wpdtc-cover-image` - 封面图生成器，五维定制系统
- `wpdtc-infographic` - 专业信息图生成器，21 布局 x 21 风格
- `wpdtc-slide-deck` - 专业幻灯片图片生成器
- `wpdtc-post-to-x` - 发布内容到 X (Twitter)
- `wpdtc-post-to-wechat` - 发布内容到微信公众号
- `wpdtc-post-to-weibo` - 发布内容到微博
- `wpdtc-post-to-wordpress` - 通过 REST API 发布内容到 WordPress

**AI 生成技能:**
- `wpdtc-imagine` - AI 图像生成，支持 11 种服务商
- `wpdtc-danger-gemini-web` - Gemini Web 交互，文本和图片生成

**工具技能:**
- `wpdtc-compress-image` - 压缩图片保持质量
- `wpdtc-format-markdown` - 格式化纯文本或 Markdown 文件
- `wpdtc-markdown-to-html` - Markdown 转样式化 HTML
- `wpdtc-translate` - 三模式翻译（快速、标准、精翻）
- `wpdtc-url-to-markdown` - 通过 Chrome CDP 抓取 URL 转 Markdown
- `wpdtc-danger-x-to-markdown` - X (Twitter) 内容转 Markdown
- `wpdtc-youtube-transcript` - 下载 YouTube 视频字幕
