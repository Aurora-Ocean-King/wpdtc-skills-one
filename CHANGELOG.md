## 1.0.0 - 2026-05-11

### Initial Release

First release of wpdtc-skills-one, a collection of 18 AI-powered skills for content generation, publishing, and utility workflows.

**Content Skills:**
- `wpdtc-article-illustrator` - Smart article illustration with Type x Style x Palette system
- `wpdtc-comic` - Knowledge comic creator with art style x tone combinations
- `wpdtc-cover-image` - Cover image generator with 5D customization
- `wpdtc-infographic` - Professional infographic generator with 21 layouts x 21 styles
- `wpdtc-slide-deck` - Professional slide deck image generator
- `wpdtc-post-to-x` - Post content to X (Twitter)
- `wpdtc-post-to-wechat` - Post content to WeChat Official Account
- `wpdtc-post-to-weibo` - Post content to Weibo
- `wpdtc-post-to-wordpress` - Post content to WordPress via REST API

**AI Generation Skills:**
- `wpdtc-imagine` - AI image generation supporting 11 providers
- `wpdtc-danger-gemini-web` - Gemini Web interaction for text and image generation

**Utility Skills:**
- `wpdtc-compress-image` - Compress images while maintaining quality
- `wpdtc-format-markdown` - Format plain text or Markdown files
- `wpdtc-markdown-to-html` - Convert Markdown to styled HTML
- `wpdtc-translate` - Three-mode translation (quick, normal, refined)
- `wpdtc-url-to-markdown` - Fetch URLs via Chrome CDP and convert to Markdown
- `wpdtc-danger-x-to-markdown` - Convert X (Twitter) content to Markdown
- `wpdtc-youtube-transcript` - Download YouTube video transcripts
